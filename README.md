## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/islamm771/portfolio.git
```

### Navigate to project folder

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
